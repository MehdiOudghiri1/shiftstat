# ShiftStat

ShiftStat is a tabular-first scientific Python library for studying reliability under distribution shift.

It provides:

- Feature-wise shift detection for mixed-type tabular data
- Dataset-level summaries and classifier two-sample tests
- Importance weighting for covariate shift correction
- Calibration analysis and post-hoc recalibration under shift
- Reliability profiles comparing reference and target domains
- Subgroup-aware degradation analysis with support-aware warnings
- Conditional auditing over confidence and shift-severity slices
- Failure-slice discovery for concentrated deployment risk
- Selective prediction and abstention analysis under deployment shift
- Risk-coverage summaries and subgroup-aware rejection diagnostics
- Reproducible benchmark scenarios and config-driven experiments
- End-to-end model evaluation workflows
- Weighted metrics, scientific reports, and publication-friendly plots

The project is designed for researchers and engineers who want transparent, reproducible tools rather than opaque dashboards. V5 extends that emphasis into publication-grade benchmarking, experiment manifests, and artifact generation for software papers and preprints.
