# Reliability Diagnostics

`shiftstat.reliability` turns predictive probabilities into structured reliability profiles across reference and target distributions.

## What it measures

- performance degradation under shift
- calibration degradation under shift
- confidence-conditioned error behavior
- uncertainty bucket summaries
- deployment fragility indicators based on explicit thresholds

## Main entry points

- `ReliabilityAnalyzer`
- `ReliabilityProfile`
- `ReliabilityShiftReport`
