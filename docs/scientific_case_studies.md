# End-To-End Scientific Case Studies

ShiftStat now supports case studies that connect multiple scientific questions in one reproducible workflow.

## Example narratives

- Covariate-shift sweeps showing when weighting or recalibration recover reliability
- Hidden subgroup failure studies where global averages miss the first operational failures
- Selective-prediction studies comparing raw acceptance, weighted threshold tuning, and recalibrated abstention
- Mixed synthetic settings exploring how dimension, imbalance, and shift pattern change conclusions

## Recommended structure for a case study

1. Define the scenario family and hypothesis.
2. Compare raw, corrected, subgroup-aware, and selective baselines.
3. Export reproducible artifacts.
4. Interpret both aggregate and disaggregated outcomes.

The examples and `paper_assets/` directory provide small but complete starting points for this workflow.
